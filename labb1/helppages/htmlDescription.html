Hur ska man använda de strukturella HTML5 beståndsdelarna

Grundstruktur

Om man tittar på webbsidor med olika innehåll och utseende hittar man en sak gemensam. De brukar delas upp i header, navigation meny, innehåll, footer och eventuellt en side bar.

Traditional HTML layouts

Innan HTML inledde de nya elementer <body> delen var indelad med <div> taggen som användes med ett beskrivande id. I den nya versionen fick dessa id, sina egna, reserverade namn.

The new HTML5 elements:

  1. <header>: För header info.
  2. <nav>: För menu eller annan navigering funktionalitet. Det brukar vara inbakad i header del.
  3. <article>: individuella delar av innehållet.
  4. <section>: Används för att gruppera artiklar enligt någon synpunkt eller för att dela upp artiklar i mindre delar.
  5. <aside>: Innehållet är relaterad till huvudinnehållet men tillhör inte tilll dess central flyt.
  6. <footer>: För footer, som brukar innehålla copyrights och snabb kontaktinfo.
